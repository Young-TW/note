導數的基本公式：

x^n / (x-a) = (x^(n-1) + ax^(n-2) + a^2x^(n-3) + ... + a^(n-1)) / (x-a)

我們可以使用導函數來求導數，並且多項式的導函數次方項與係數可以被拆開，每個單項無係數（已被拆開）導函數的計算方式如下：

如果 f(x) = x^n， 則 f'(x) = nx^(n-1)

所以 f(x) = x^3 + 2x^2 + 3x + 4 的導函數為 f'(x) = 3x^2 + 4x + 3
