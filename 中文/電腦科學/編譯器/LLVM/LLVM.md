# LLVM

LLVM 是一套編譯器的工具鏈，
LLVM 用一套系統化的編譯方式實現了許多不同語言的編譯器，
LLVM 將編譯工作用 LLVM-IR 拆解成三個部分，
分別為前端、最佳化以及後端。  

## LLVM 前端

LLVM 前端需要將原始碼拆解成通用的 LLVM-IR，
生成抽象語法樹 AST，
再交給最佳化和後端處理剩下的工作。  

## LLVM-IR 最佳化

最佳化的工作非常困難，
首先要取得前端處理完語法分析並生成的 LLVM-IR，
接著分析程式碼結構以及工作，
並且找到等效卻更高效能的替代方案，
產生出最佳化完成後的 LLVM-IR。  

## LLVM 後端

後端負責將最佳化後的 LLVM-IR 轉換為各種不同平台的二進位可執行檔。
