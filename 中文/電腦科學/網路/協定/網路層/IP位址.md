# IP 位址

[TOC]

## 簡介

IP 位址就像是電腦在網路上的門牌號碼，
收發資料都會使用到 IP 位址，
IP 位址分為 IPv4 和 IPv6，
以下會分別介紹。  

## IPv4

IPv4 是在電腦開始發展後不久所訂下的，
由四個 0~255 的數組構成，
例如以下 IP：  

`155.274.36.175`

就是一組普通的 IP 位址。  

在區域網路中會有特殊的 IPv4 位址保留給區域網路中的設備使用，
詳細內容請參考 [區域網路](區域網路.md)。  

### 問題

IPv4 能夠分配的總 IP 數量侷限在 `256^4`，
已經出現了 IP 數量不足的情形，
因此 IPv6 也就誕生了。
