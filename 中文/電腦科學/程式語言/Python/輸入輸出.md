# Python 輸入輸出 (input output)

[TOC]

### input

#### 語法

```python=
變數 = input()
```

#### 範例

```python=
a = input()
```

#### 解說

將輸入值指定到變數``a``  
``a``的資料型態會交由Python自動分配  

### print

#### 語法

```python=
print("內容")
print(變數)
```

#### 範例

```python=
a=1
print("Hello world!")
print(a)
print(f"Hello world{a}")
```

#### 結果

```
Hello world!
1 
Hello world1
```  

#### 解說

第一行印出字串``Hello world!``  
第二行印數變數``a``  
第三行印出格式化字串``Hello world1``
