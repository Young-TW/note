## 語法

以下拿現實中的動物作為範例，
每當新增一隻動物就會需要該種動物的類別，
加上所有動物都有年齡的特性，
所以可以宣告一個名為 animal 的類別。  

```cpp
class animal{
    public:
        int age;
        int weight;
        int height;
        string name;
};

class cat : public animal{
    public:
        int jump_height;
};

class dog : public animal{
    public:
        int bark_db;
};
```
