# C++ 迴圈

[TOC]

## For 迴圈

### 基礎的 For 迴圈

```cpp
for(int i=0;i<10;i++){
    
}
```

### Auto(vector)

```cpp
vector<int> n;
for(auto i : n){
    
}
```

### 無限迴圈

```cpp
for(;;){

}
```

## While 迴圈

### 特性

While 迴圈會在每一圈開始以前會參考小括號內的布林值，
如果小括號內為 False 就不會進入迴圈，
反之如果小括號內為 True 就會進入迴圈，
並且在迴圈結束後重新判斷一次小括號內的值。  

### 使用方式

當條件成立時
```cpp
while(/*條件*/){

}
```

### 無限迴圈

```cpp
while(1){

}

// or

while(True){

}
```

## 特例

### break:

#### 簡介

跳脫迴圈，直接從迴圈結束的地方繼續執行程式。

#### 語法

```cpp
break;
```

### continue:

#### 簡介

無視後面的內容，直接從迴圈下一圈開頭繼續執行程式。

#### 語法

```cpp
continue;
```
