C++在宣告變數時需要指定資料型態  
int(整數)和數學的整數定一一樣  
short是比較短的整數  
消耗的記憶體空間也比較少  
long是比較長的整數  
消耗的記憶體空間比較多  
float(浮點數)小數點後有還有非0的數  
double是比較長的浮點數  
消耗的記憶體空間比float更多  
bool(布林值)是或否 也就是1或0  
char(字元)可以存一個字  

### Example

```cpp=
int a;
```
