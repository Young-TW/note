## 和差角公式

1. 和差角公式  

已知 α 、 β 為任意兩角：  

正弦函數：  
sin(α+β) = sinα cosβ + cosα sinβ。  
sin(α-β) = sinα cosβ - cosα sinβ。  

餘弦函數：  
cos(α+β) = cosα cosβ + sinα sinβ。  
cos(α-β) = cosα cosβ - sinα sinβ。  

正切函數：  
tan(α+β) = (tanα + tanβ) / (1 - tanα tanβ)，其中tanα tanβ != 1。  
tan(α-β) = (tanα - tanβ) / (1 + tanα tanβ)，其中tanα tanβ != -1。

cos15° = (√6 - √2) / 4 = sin75°
