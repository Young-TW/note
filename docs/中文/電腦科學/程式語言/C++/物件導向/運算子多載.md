# C++ 運算子多載 (operator overloading)

[TOC]

## 簡介

運算子多載可以用來做合併兩個物件之類原本 `C++` 沒有支援的功能  

## 語法

```cpp
class Ball{
    public:
        int weight;
        Ball()
        Ball(int n): weight(n){}

        Ball operator+(Ball &obj){
            Ball res;
            res.weight = this -> obj.weight;
            return res
        }
};
```
