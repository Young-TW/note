# C++ 模板 (template)

[TOC]

## 簡介

現有的函式不管是參數還是回傳值都必須明確指定資料型態，
這讓很多函式都被重複撰寫，
而模板的出現解決了這個問題。  

## 語法

以下使用 `add()` 函式來作範例：  

```cpp
template<class T>

T add(T a, T b){
    return a+b;
}
```

如果不使用模板就會像下面這樣：  

```cpp
int add(int a, int b){
    return a+b;
}

double add(int a, int b){
    return a+b;
}

long add(int a, int b){
    return a+b;
}
```

重複撰寫函式不僅讓程式碼更加混亂且容易發生錯誤。

模板也可以一次使用兩種以上不同型態作為輸入輸出：

```cpp
template<class T>
template<class U>

T add(T a; U b){
    return a+b;
}
```
