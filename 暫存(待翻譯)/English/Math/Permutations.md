## 階乘

階乘定義
```
n! = 1*2*3*.....*n

(n-n)! = 0! = 1

n! = n*(n-1)!
```

## 排列

### 公式

### 完全相異物的直線排列

- P N取K (排列)

自n個完全相異物中任取k個作排列
```
n! / (n-k)!
```

### 不盡相異物的直線排列

```
(n+m)! / n!*m!
```

## 組合
不考慮順序

### 公式
- C N取K (組合)

```
n! / (n-k)!*k!
```

1. C N取N = 1
2. C N取0 = 1
3. C N取K = C N取(N-K)
4. C N取K = (C N-1取K-1) + (C N-1取K)
5. C N取0 + C N取1 + C N取2 + ... C N取N = 2^n
6. C N取0 + C N取2 + C N取4 + ... C N取N 
= C N取1 + C N取3 + C N取5 + ... C N取N 
= 2^n-1
